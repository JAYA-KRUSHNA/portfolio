## ✨ Features

- **3D Interactive Background** — Smooth multi-color particle field powered by Three.js
- **Custom Cursor** — Magnetic ring cursor with hover and click animations
- **Typed Text Animation** — Cycles through role descriptions in the hero section
- **Glassmorphism UI** — Dark premium cards with backdrop blur and glow effects
- **3D Card Tilt** — Profile card responds to mouse movement with perspective rotation
- **Scroll Reveal** — Elements animate in as you scroll
- **Ripple Click Effects** — Every interactive element has satisfying feedback
- **Active Nav Highlighting** — Navbar link tracks the current visible section
- **Animated Stat Counters** — Numbers count up when scrolled into view
- **Fully Responsive** — Works on desktop, tablet, and mobile

## 🗂️ Sections

| Section | Description |
|---|---|
| **Hero** | Animated name, typed roles, 3D card, stat counters |
| **About** | Bio, education info, leadership roles with SVG icons |
| **Skills** | Programming languages with progress bars, tech stack bubbles |
| **Projects** | SLEVIS, BookSync, OptiSchedule — with GitHub links |
| **Achievements** | Timeline with SVG icons & rank badges, certifications panel |
| **Contact** | Email, GitHub, LinkedIn cards with animated arrows |

## 🛠️ Tech Stack

- **HTML5** — Semantic structure
- **CSS3** — Custom properties, Grid, Flexbox, animations
- **Vanilla JavaScript** — All interactions, no framework needed
- **Three.js (r134)** — 3D particle background
